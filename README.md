Impnattypo
==========

[![CI](https://github.com/raphink/impnattypo/actions/workflows/ci.yml/badge.svg)](https://github.com/raphink/impnattypo/actions/workflows/ci.yml)
[![CTAN](https://img.shields.io/ctan/v/impnattypo.svg)](https://www.ctan.org/pkg/impnattypo)

This package provides useful macros implementing recommendations by the French
Imprimerie Nationale. The documentation of this package is in French.

Ce paquet fournit des macros utiles implémentant des recommandations de
l'Imprimerie Nationale Française.
